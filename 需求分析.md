# 📋 凑单折扣搭配App - 需求分析文档

---

## 一、项目概述

### 1.1 项目背景
本App旨在帮助用户在购物时通过智能算法找到最优的凑单方案和优惠券使用策略，实现最低总价购物。

### 1.2 核心目标
- **省钱**：通过智能算法计算最优购物方案
- **便捷**：简单输入商品价格和优惠券规则
- **智能**：自动推荐凑单商品，实现"差一点满减"场景的最优解

## 二、功能需求分析

### 2.1 商品管理模块

| 需求点 | 描述 | 优先级 |
|--------|------|--------|

| 商品分类 | 支持商品分类管理 | ⭐⭐ |

### 2.2 优惠券管理模块

| 需求点 | 描述 | 优先级 |
|--------|------|--------|
| 添加优惠券 | 输入优惠类型、门槛、折扣值 | ⭐⭐⭐ |

## 三、业务场景分析

### 3.1 场景一：普通满减优惠
- **输入**：商品总价¥280，满200减30券、满100减20券
- **计算**：自动选择最优组合（满200减30 + 满100减20）
- **输出**：最终价格¥230

### 3.2 场景二：不可叠加优惠券
- **输入**：商品总价¥280，店铺8折券（不可叠加）、平台满200减30券（可叠加）
- **计算**：分别计算使用8折券、使用满减券、同时使用的情况
- **输出**：选择最优方案

### 3.3 场景三：差一点满减（核心场景）
- **输入**：商品总价¥99.9，满100减20券
- **计算**：检测差距¥0.1，推荐凑单商品
- **输出**：推荐购买纸巾¥1.5，最终价格¥81.4，净节省¥18.5

### 3.4 场景四：多张满减券叠加
- **输入**：商品总价¥350，满100减20券（可重复使用）
- **计算**：自动计算可使用3张，共减¥60
- **输出**：最终价格¥290

## 四、非功能需求

### 4.1 性能需求
- 算法计算响应时间 < 1秒
- 支持最多50件商品和20张优惠券的计算

### 4.2 用户体验
- 简洁直观的操作界面
- 清晰的方案对比展示
- 友好的凑单建议提示

### 4.3 数据管理
- 本地数据持久化存储
- 预置常用凑单商品数据

## 五、数据模型概览

### 5.1 商品模型
```
Product {
    id: Long
    name: String
    originalPrice: Double
    quantity: Int
}
```

### 5.2 优惠券模型
```
Coupon {
    id: Long
    name: String
    type: CouponType (FULL_REDUCTION/DISCOUNT/NO_THRESHOLD)
    threshold: Double      // 满减门槛
    discountValue: Double  // 折扣值
    maxUsages: Int         // 最大使用次数
    isStackable: Boolean   // 是否可叠加
    isSingleUse: Boolean   // 是否仅限单次使用
}
```

### 5.3 凑单商品模型
```
FillProduct {
    id: Long
    name: String
    price: Double
    category: String
    isHot: Boolean
}
```

### 5.4 凑单方案模型
```
BundleSolution {
    products: List<Product>
    couponUsages: List<CouponUsage>
    originalTotal: Double
    finalPrice: Double
}

CouponUsage {
    coupon: Coupon
    count: Int  // 自动计算的使用数量
}
```

### 5.5 凑单建议模型
```
FillSuggestion {
    fillProducts: List<FillProduct>
    originalTotal: Double
    newTotal: Double
    discount: Double
    finalPrice: Double
    saving: Double  // 净节省金额
}
```

## 七、总结

本App核心价值在于：

1. **智能算法**：自动计算最优优惠券使用数量和组合
2. **场景覆盖**：支持可叠加/不可叠加优惠券、差一点满减等多种场景
3. **用户友好**：只需输入规则，无需手动计算
4. **实用推荐**：智能推荐低价凑单商品，最大化节省

这是一个实用性强、算法核心明确的工具类App，能够切实帮助用户在购物时省钱。
